# Python Galgame Engine (Learning Project)

这是一个基于 Python 开发的轻量级恋爱养成游戏（Galgame）引擎框架。本项目的主要目的是进行 **Python 编程练习** 与 **面向对象设计（OOP）** 的实践学习。

> **声明**：本项目目前处于早期开发阶段，仅用于个人代码练习与逻辑实现。功能非常简陋，但后续会持续更新并完善功能。

------

## 🏗 项目架构

项目采用了模块化的面向对象设计，核心文件及其职责如下：

- **`character.py`**: 角色系统。定义了基础角色类、可攻略女主角类及其好感度（Affinity）计算逻辑。
- **`game.py`**: 引擎核心。负责管理游戏循环、剧情流转、共通线与个人线的切换，以及存档/读档机制。
- **`pool.py`**: 数据工厂。用于统一实例化所有游戏角色，解耦业务逻辑与具体数据。
- **`check.py`**:检查文件配置是否完整
- **`save/`**: 存储目录。存放 JSON 格式的玩家存档数据。
- **`story/`**: 剧本目录。使用 JSON 格式存储结构化的剧情对白。

------

## ✨ 已实现功能

- **面向对象继承体系**: 通过基类 `Character` 派生出 `MainCharacter` 和 `NPC`，实现了良好的代码复用。
- **灵活的存档系统**: 支持多档位存储，记录包括当前剧情位置（`screen_idx`）、角色好感度以及是否进入个人线等状态。
- **自动化剧本加载**: 能够解析 JSON 格式的剧情文件，并根据角色姓名动态匹配个人线内容。
- **好感度判定**: 实时追踪角色好感度，并在共通线结束时判定是否达到进入个人线的阈值（$Affinity \ge 100$）。
- **剧情跳过功能**：可记录历史已读，可直接跳过共通线已读剧情以及角色线已读剧情，并支持直接跳跃到下一个选项
